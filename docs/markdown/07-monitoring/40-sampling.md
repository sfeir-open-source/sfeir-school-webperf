# Observabilité

## La segmentation des données

Il est primordial de récolter un maximum de contexte pour pouvoir segmenter correctement les données :

- La page
- Le type de device (mobile / desktop...)
- l'OS
- Le navigateur
- L'authentification
- L'état du consentement aux cookies
- Les feature flags
- Les A/B tests
  ...
