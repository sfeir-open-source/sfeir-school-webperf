# 6. La compression

## Les algorithmes de compression

- Ne conserve pas la syntaxe
- Nécessite d'être dé-compressé
- Plusieurs algorithmes possibles : Brotli, Gzip ...
- Le flux de compression / décompression n'est pas bloquant
