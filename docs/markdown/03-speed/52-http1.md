# 5. Protocole HTTP

## HTTP/1 - 1996

😍 Avancées :

- Permet la navigation hypertexte basique
- Concept de requêtes/réponses via TCP
- Premiers verbes HTTP : GET, POST, et HEAD

<div>

😭 Limitations :

- Une nouvelle connexion TCP est établie pour chaque requête
- Performance faible
- Pas de compression

</div>
